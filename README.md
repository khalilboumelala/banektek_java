**System Requirement :**
- Java JDK 17.0.12
- MYSQL
- Java FX
  
**System Features:**

## How to install and start the application:

- To install all the dependencies:

            mvn clean install
            
- To start application using mvn command:

            mvn compile exec:java
           
- To create executable jar file in shade directory:

            mvn compile package
